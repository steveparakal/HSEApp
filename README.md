# HSEApp

An all-in-one Android application built for university students and professors, streamlining academic life through timetable management, assignments, grading, online class integration, and real-time communication.

---

## Overview

HSEApp is a university project developed in **Kotlin** for Android. It serves as a centralized platform for both students and faculty, replacing the need for multiple disconnected tools by bringing core academic features into a single, easy-to-use mobile app.

---

## Features

- **Timetable Viewer** — Browse your weekly class schedule at a glance.
- **Assignment Management** — View upcoming assignments, deadlines, and submission status.
- **Grading Tools** — Professors can manage and submit grades directly from the app.
- **Zoom Integration** — Access integrated Zoom links for online classes without switching apps.
- **Subject-Based Chat** — Communicate with students or faculty within dedicated subject channels.

## Getting Started

### Prerequisites

- [Android Studio](https://developer.android.com/studio) (latest stable recommended)
- Android SDK (API level as configured in `build.gradle`)
- Java 11 or higher

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/steveparakal/HSEApp.git
cd HSEApp
```

2. **Open in Android Studio:**
   - Launch Android Studio
   - Select **File → Open** and navigate to the cloned folder

3. **Sync Gradle:**
   - Android Studio will prompt you to sync — click **Sync Now**

4. **Run the app:**
   - Connect an Android device or start an emulator
   - Click the **Run** button (▶) or use `Shift + F10`
